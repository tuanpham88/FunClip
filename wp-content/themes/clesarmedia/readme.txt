==== Installation ====
Manual installation:

1. Upload the `ClesarMedia` folder to the `/wp-content/themes/` directory
== or ==
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory
2. Go into your WordPress Admin, navigate to 'Appearance > Themes'
3. Find the Big City listing on this page and click 'Activate'

== Activiation and Use ==
1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options
